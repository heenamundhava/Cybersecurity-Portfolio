# Nmap

## Description
Nmap is a network scanning tool used to discover hosts and services on a network.

## Common Commands

nmap target_ip

nmap -sS target_ip

nmap -sV target_ip

## What It Does
- Identifies open ports
- Detects services running on a system
- Helps identify potential vulnerabilities

## Use in Cybersecurity
Nmap is commonly used in penetration testing and security assessments to understand network exposure.
