# TryHackMe – Nmap Scanning

## Tool
Nmap

## Objective
Understand network scanning techniques used to discover open ports and services.

## Commands Used
nmap -sS target_ip
nmap -sV target_ip

## Key Learnings
- Identifying open ports
- Detecting running services
- Understanding attack surface

## SOC Relevance
Network scanning helps security analysts detect vulnerabilities and unauthorized services running on systems.
