# TryHackMe – Linux Fundamentals

## Objective
Learn basic Linux commands used in cybersecurity environments.

## Commands Practiced
- ls
- cd
- pwd
- cat
- grep
- chmod
- sudo

## Key Learnings
- Navigating directories
- Viewing and editing files
- Understanding Linux permissions
- Using command line tools for investigation

## SOC Relevance
Linux servers are widely used in production environments. SOC analysts must understand Linux systems to analyze logs and detect suspicious activity.
